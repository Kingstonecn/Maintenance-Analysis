# 维修数据分析 · 使用说明

## 1. 打开工具

用浏览器打开 `Maintenance-Analysis.html` 即可，无需安装任何软件。推荐使用 Chrome 或 Edge。

## 2. 导入数据

将维修单 Excel 文件（.xlsx / .xlsm）拖入页面中央的虚线框，或点击「选取文件」按钮选择文件。

- 文件在浏览器本地解析，不会上传到任何服务器。
- 工具会自动读取第一个工作表并识别 57 列字段。

## 3. 数据概览

导入后页面显示数据概览：

- **统计条**：维修单总数、设备数量、车间数量、日期范围。
- **数据概览图表**（可折叠）：维修状态分布、车间分布、车间分类、紧急程度分布、班次分布、故障等级分布。
- **数据预览**（可折叠）：前 20 行数据预览。

点击「重新载入」可返回导入页面更换文件。

## 4. 分析面板

页面下方为分析面板，共 81 种分析方法。

- **左侧**：方法浏览器，可搜索、按分析维度和综合评分筛选。
- **右侧**：点击任意方法后展示分析结果（图表 + 表格）。

## 5. 导出分析

点击右上角「导出分析」按钮，选择导出格式和范围：

- **PDF**：每个分析占一页，含图表和表格。
- **Excel**：每个分析占一个 Sheet，含表格数据和图表。
- **PDF + Excel**：同时导出两份文件。

导出范围可选全部方法、综合评分 5 分、或 4 分及以上。

## 6. 常见问题

| 问题 | 解决方法 |
|------|----------|
| 导入后没有反应 | 确认文件格式为 .xlsx 或 .xlsm，且第一个工作表包含数据 |
| 图表不显示 | 刷新页面后重新导入 |
| 导出速度慢 | 属正常现象，81 种方法逐一渲染需要一些时间 |
| 导出文件打不开 | 确认浏览器未拦截文件下载 |
