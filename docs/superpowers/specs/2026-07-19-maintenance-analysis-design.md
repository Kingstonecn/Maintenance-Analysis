# 维修数据分析 — 分析方法设计

> **项目**: Maintenance-Analysis  
> **日期**: 2026-07-19  
> **模式**: 纯前端（浏览器本地处理），参考 AEData-Check 项目  
> **数据源**: Maintenance Data.xlsx（8648 条维修记录，57 列字段）

---

## 1. 项目概述

### 1.1 目标

对维修工单数据进行全面分析，覆盖设备可靠性、维修效率、成本影响、评分质量等维度，提供从简单描述统计到预测分析的 74 种分析方法，供用户筛选出理想的分析方法组合。

### 1.2 技术模式

- **纯前端**：所有处理在浏览器本地完成，文件不离开本机
- **本地加载**：拖拽/选择 Excel 文件，SheetJS 解析
- **数据分析**：74 种方法，按维度+复杂度组织
- **图表导出**：CSV 数据导出 + 图表截图导出

## 2. 分析方法组织方案

### 2.1 组织方式

**维度分类 + 复杂度标签**（方案 C）

- 按业务维度分 10 组
- 每个方法标注复杂度：⭐简单 / ⭐⭐中等 / ⭐⭐⭐复杂 / ⭐⭐⭐⭐预测
- 用户可按维度浏览，也可按复杂度筛选

### 2.2 复杂度定义

| 等级 | 含义 | 特征 |
|------|------|------|
| ⭐ | 简单 | 描述统计、计数、分布，一键生成无需配置 |
| ⭐⭐ | 中等 | 需选择维度或时间粒度，涉及分组聚合 |
| ⭐⭐⭐ | 复杂 | 涉及统计建模（回归/相关/MTBF等），可能需配置参数 |
| ⭐⭐⭐⭐ | 预测 | 涉及时间序列预测或概率模型 |

## 4. UI 架构

### 4.1 页面流程

```
加载页面 → 数据概览（已实现）→ 分析面板
                                    ↓
                          ┌─── 分析方法浏览器 ───┐
                          │  维度筛选 + 复杂度筛选  │
                          │  74个方法卡片列表      │
                          └────────┬───────────┘
                                   ↓
                          ┌─── 分析执行区 ───┐
                          │  方法配置(参数)   │
                          │  图表渲染         │
                          │  数据表格         │
                          │  导出按钮         │
                          └─────────────────┘
```

### 4.2 分析面板布局

**左侧：方法浏览器（可折叠）**
- 顶部：维度筛选标签（全部/①~⑩）+ 复杂度筛选（⭐/⭐⭐/⭐⭐⭐/⭐⭐⭐⭐）
- 搜索框：按关键词搜索方法
- 下方：方法卡片列表，每卡显示：编号、名称、复杂度星标、涉及字段、一句话说明

**右侧：分析执行区**
- 选中方法后显示：
  - 方法标题 + 复杂度 + 涉及字段
  - 配置区（如需选择维度/阈值等参数）
  - 「执行分析」按钮
  - 图表区（Canvas/SVG 渲染）
  - 数据表格（可排序）
  - 「导出图表」/「导出CSV」按钮

### 4.3 技术选型

| 组件 | 方案 | 来源 |
|------|------|------|
| 图表渲染 | Chart.js | CDN |
| 统计计算 | simple-statistics | CDN |
| 文本分析 | 纯 JS 分词 + 词频统计 | 内置 |
| 数据导出 | CSV (Blob) + 图表截图 | 内置 |
| 预测分析 | 纯 JS 移动平均/指数平滑/线性回归 | 内置 |

## 6. 数据流

```
Excel 文件
  ↓ SheetJS 解析
S.rows[] (JSON 对象数组，按列字母索引)
  ↓ computeStats()
S.stats (各维度计数)
  ↓ showOverview()
数据概览页面（状态/车间/班次等分布图）
  ↓ 用户选择分析方法
analysis.js 中对应函数(rows, config) → {chartData, tableData}
  ↓ charts.js 渲染
图表 + 表格
  ↓ 用户点击导出
CSV / PNG 下载
```
