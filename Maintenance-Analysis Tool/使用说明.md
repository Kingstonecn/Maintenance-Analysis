# 维修数据分析 · 使用说明

> **纯浏览器端工具** · 无需安装 · 数据不离开本机

---

## 1. 打开工具

用浏览器打开 `Maintenance-Analysis.html` 即可。**推荐 Chrome 或 Edge**，无需安装任何插件。

---

## 2. 导入数据

将维修单 Excel 文件拖入页面中央的虚线框，或点击 **「选取文件」** 按钮选择。

- **支持格式**：`.xlsx` / `.xlsm`
- **数据安全**：文件在浏览器本地解析，**不会上传到任何服务器**
- 自动读取第一个工作表，识别 **57 列字段**（维修单编号、状态、车间、设备、故障、维修时长等）

> ⚠️ 确认第一个工作表包含数据，否则导入后无反应。

---

## 3. 数据概览

导入后页面顶部显示统计条，下方为可折叠的概览区域：

| 区域 | 内容 |
|------|------|
| **统计条** | 维修单总数 · 设备数量 · 车间数量 · 日期范围 |
| **数据概览图表** | 维修状态分布 · 车间分布 · 车间分类 · 紧急程度 · 班次 · 故障等级 |
| **数据预览** | 前 20 行原始数据 |
| **智能分析建议** | 自动生成的改进建议（见第 5 节） |

点击 **「重新载入」** 可返回导入页面更换文件。

---

## 4. 分析面板

页面下方为分析面板，共 **81 种分析方法**，涵盖 11 个维度：

| 维度 | 示例方法 |
|------|----------|
| ① 描述性统计 | 维修单总量、时长统计、费用统计 |
| ② 排名与帕累托 | 设备故障帕累托、故障原因帕累托 |
| ③ 时间趋势 | 月度趋势、班次月度趋势 |
| ④ 设备可靠性 | MTBF、MTTR、可用度、重复故障 |
| ⑤ 维修效率 | 等待占比、班次效率、维修人对比 |
| ⑥ 成本与影响 | 成本分布、备件 vs 委外、产能影响 |
| ⑦ 评分与质量 | 评分分布、评分排名、车间评分 |
| ⑧ 相关性分析 | 频率 vs 成本、人数 vs 时长 |
| ⑨ 回归与预测 | 时长回归、成本回归、ARIMA 预测 |
| ⑩ 文本分析 | 故障描述词频、TF-IDF 关键词 |
| ⑪ 大数据分析 | 关联规则、K-Means 聚类、季节性分解 |

### 操作方式

- **左侧方法浏览器**：搜索框输入关键词，或按 **分析维度** 和 **综合评分**（★★★ 3.0+ / ★★★½ 3.5+ / ★★★★ 4.0+ / ★★★★½ 4.5+）筛选
- **右侧分析结果**：点击任意方法后展示 **图表 + 表格 + 分析结论**
- **数据筛选**：在方法浏览器上方可按 **年月** 和 **车间** 筛选数据范围
- **收藏功能**：点击方法卡片上的 ♥ 可收藏常用方法

> 💡 **综合评分** = 业务价值 × 60% + 易用性 × 40%，用于快速找到高价值分析方法。

---

## 5. 智能分析建议

导入数据后，系统自动生成智能分析建议，位于数据概览下方。

- **自动分析**：基于全局和各车间维度交叉分析
- **严重程度**：🔴 高优先级 → 🟡 中等 → 🟢 低优先级
- **可跳转**：每条建议右侧 **ℹ️** 图标可点击展开，显示关联的分析方法，**点击方法名可直接跳转**到对应分析
- **综合诊断**：当多个问题同时出现时，系统会生成综合诊断建议（如"设备老化严重"、"预防性维护不足"等）

---

## 6. 导出报告

点击右上角 **「导出分析」** 按钮，选择导出格式：

| 格式 | 内容 |
|------|------|
| **PDF** | 封面 + 智能建议 + 各方法（含字段 / 方法说明 / 分析结论 / 图表 / 表格） |
| **Excel** | 封面 + 目录跳转页 + 各方法 Sheet（含字段 / 方法说明 / 分析结论 / 表格） |
| **PDF + Excel** | 同时导出两份文件 |

导出前可按 **年月** 和 **车间** 筛选数据范围。

> ⚠️ 导出 81 种方法需要逐一渲染，**耗时较长属正常现象**，请耐心等待。

---

## 7. 常见问题

| 问题 | 解决方法 |
|------|----------|
| 导入后没有反应 | 确认文件格式为 **.xlsx 或 .xlsm**，且第一个工作表包含数据 |
| 图表不显示 | **刷新页面**后重新导入 |
| 导出速度慢 | 属正常现象，81 种方法逐一渲染需要一些时间 |
| 导出文件打不开 | 确认浏览器**未拦截文件下载** |
| 建议中 ℹ️ 面板不显示 | 点击 ℹ️ 图标后面板出现在图标下方，点击 **✕** 或页面空白处关闭 |
